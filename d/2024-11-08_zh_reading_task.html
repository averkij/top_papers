OpenCoder: The Open Cookbook for Top-Tier Code Large Language Models

1. 这篇文章讨论了大型语言模型（LLMs）在代码生成、推理任务和代理系统中的重要性。

2. 虽然开放访问的代码LLMs性能接近专有模型，但适合严谨科学研究的高质量代码LLMs仍然有限。

3. 为了填补这一空白，文章介绍了OpenCoder，一个顶尖的代码LLMs，其性能堪比领先模型，并且为研究社区提供了一个“开放的食谱”。

4. OpenCoder不仅发布了模型权重和推理代码，还包括可重复的训练数据、完整的数据处理流水线、严格的实验消融结果和详细的训练协议，以促进科学研究的开放和可重复性。

1. Zhè piān wénzhāng tǎolùn le dàxíng yǔyán móxíng (LLMs) zài dàimǎ shēngchéng, tuīlǐ rènwù hé dàilǐ xìtǒng zhōng de zhòngyàoxìng

4. OpenCoder bùjǐn fābùle móxíng quánzhòng hé tuīlǐ dàimǎ, hái bāokuò kě chóngfù de xùnliàn shùjù, wánzhěng de shùjù chǔlǐ liúshuǐxiàn, yánjìn de shìyàn róngróng jiéguǒ hé xiángxì de xùnliàn xiéyì, yǐ cùjìn kēxué yánjiū de kāifàng hé kě chóngfùxìng

1. This article discusses the importance of large language models (LLMs) in code generation, reasoning tasks, and agent systems.

2. Although open-access code LLMs perform nearly as well as proprietary models, high-quality code LLMs suitable for rigorous scientific research remain limited.

3. To fill this gap, the article introduces OpenCoder, a top-tier code LLM with performance comparable to leading models, and provides an "open recipe" for the research community.

4. OpenCoder not only releases model weights and inference code but also includes reproducible training data, a complete data processing pipeline, rigorous experimental ablation results, and detailed training protocols to promote openness and reproducibility in scientific research.

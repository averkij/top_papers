Web-Shepherd: Advancing PRMs for Reinforcing Web Agents

1. 这篇文章讨论了网页导航的自动化任务。

2. 过去的研究使用多模态大语言模型（MLLM）作为奖励模型，但这对实际应用有限制。

3. 作者提出了第一个过程奖励模型（PRM），称为Web-Shepherd，用于逐步评估网页导航路径。

4. 他们创建了一个大规模数据集和评估基准，并在实验中证明了Web-Shepherd的有效性和成本效益。

5. 所有模型、数据集和代码都公开可用。

1. This article discusses the automation of web page navigation tasks.

2. Previous research has used multimodal large language models (MLLMs) as reward models, but this has limitations for practical applications.

3. The authors propose the first process reward model (PRM), called Web-Shepherd, for step-by-step evaluation of web page navigation paths.

4. They created a large-scale dataset and evaluation benchmark, and demonstrated the effectiveness and cost-efficiency of Web-Shepherd in experiments.

5. All models, datasets, and code are publicly available.

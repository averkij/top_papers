1. This article discusses the challenges faced by modern large language models (LLMs) when handling very long contexts, including slower inference speeds and increased memory costs.

2. Most pre-trained LLMs cannot generalize beyond their original training sequence lengths.

3. To address this issue, the authors propose InfiniteHiP, a new LLM inference framework that accelerates processing by dynamically eliminating irrelevant context tokens.

4. This method also allows for generalization to longer sequences by selectively applying various RoPE adjustment methods.

5. Additionally, the authors offload key-value caches to host memory during inference, significantly reducing GPU memory pressure.

6. As a result, InfiniteHiP can process up to 3 million tokens on a single L40s 48GB GPU without permanently losing context information and achieves an 18.

7. 95-fold speedup in attention decoding within a context of 1 million tokens.

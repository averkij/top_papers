1. This article discusses the advancements of Multi-Agent Reinforcement Learning (MARL) in addressing cooperative and competitive problems.

2. The main challenge of MARL is the need to explicitly predict agent behaviors to achieve cooperation.

3. To address this issue, the authors propose the Shared Recurrent Memory Transformer (SRMT), which aggregates and globally broadcasts individual working memories, enabling agents to implicitly exchange information and coordinate actions.

4. SRMT performs well in partially observable multi-agent path planning problems and the POGEMA benchmark task set, particularly in sparse reward scenarios.

Kimi-VL Technical Report

1. 我们介绍了Kimi-VL，一种高效的开源混合专家（MoE）视觉-语言模型（VLM）。

2. 它在多模态推理、长上下文理解和强大的代理能力方面表现出色，激活的语言解码器参数仅为2.8B。

3. Kimi-VL在挑战性领域表现强劲，包括多轮代理任务、图像和视频理解、OCR、数学推理和多图像理解。

4. 它与先进的高效VLM竞争，并在几个关键领域超越GPT-4o。

5. Kimi-VL还在处理长上下文和清晰感知方面取得进展，并引入了Kimi-VL-Thinking，一种具有长链思维能力的变种模型。

6. 代码和模型可在https://github.com/MoonshotAI/Kimi-VL公开访问。

1. We introduce Kimi-VL, an efficient open-source hybrid Mixture of Experts (MoE) Vision-Language Model (VLM).

2. It excels in multimodal reasoning, long-context understanding, and strong agent capabilities, with only 2.

3. 8B activated language decoder parameters.

4. Kimi-VL performs strongly in challenging domains, including multi-turn agent tasks, image and video understanding, OCR, mathematical reasoning, and multi-image understanding.

5. It competes with advanced efficient VLMs and outperforms GPT-4o in several key areas.

6. Kimi-VL has also made progress in handling long contexts and clear perception, introducing Kimi-VL-Thinking, a variant model with long-chain thinking capabilities.

7. The code and model are publicly accessible at https://github.

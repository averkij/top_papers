1. We introduce Kimi-VL, an open-source hybrid expert visual-language model.

2. It features advanced multimodal reasoning, long-context understanding, and powerful agent capabilities, activating only 2.

3. 8B language decoder parameters.

4. Kimi-VL performs excellently in multi-turn agent tasks and various visual-language tasks.

5. It can also handle long-context and high-resolution visual inputs.

6. Based on Kimi-VL, we developed Kimi-VL-Thinking, which has strong long-term reasoning capabilities.

7. The code and model are available at https://github.

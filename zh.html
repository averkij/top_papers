OpenCoder: The Open Cookbook for Top-Tier Code Large Language Models

1. 这篇文章讨论了大型语言模型（LLMs）在代码生成、推理任务和代理系统中的重要性。

2. 虽然开放访问的代码LLMs性能接近专有模型，但适合严格科学研究的高质量模型仍然有限。

3. 为了填补这一空白，作者介绍了OpenCoder，一个顶尖的代码LLM，性能媲美领先模型，并提供详细的训练数据和协议。

4. 通过这种开放性，作者希望加速代码AI的研究和可重复的进展。

1. Zhè piān wénzhāng tǎolùn le dàxíng yǔyán móxíng (LLMs) zài dàimǎ shēngchéng, tuīlǐ rènwù hé dàilǐ xìtǒng zhōng de zhòngyàoxìng

2. Suīrán kāifàng fǎngwèn de dàimǎ LLMs xìngnéng jìnkè zhuānyǒu móxíng, dàn shìhé yánge kēxué yánjiū de gāo zhìliàng móxíng réngrán yǒuxiàn

3. Wèile tiánbǔ zhè yī kòngbái, zuòzhě jièshào le OpenCoder, yīgè dǐngjiān de dàimǎ LLM, xìngnéng jìmǐ lǐngxiān móxíng, bìng tígōng xiángxì de xùnliàn shùjù hé xiéyì

4. Tōngguò zhè zhǒng kāifàngxìng, zuòzhě xīwàng jiāsù dàimǎ AI de yánjiū hé kě chóngfù de jìnbù

1. This article discusses the importance of large language models (LLMs) in code generation, reasoning tasks, and agent systems.

2. While open-access code LLMs perform nearly as well as proprietary models, high-quality models suitable for rigorous scientific research remain limited.

3. To fill this gap, the authors introduce OpenCoder, a top-tier code LLM that matches the performance of leading models and provides detailed training data and protocols.

4. Through this openness, the authors aim to accelerate research and reproducible progress in code AI.

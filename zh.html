SAM2Long: Enhancing SAM 2 for Long Video Segmentation with a Training-Free Memory Tree

1. 这篇文章介绍了一种改进的视频对象分割策略，称为SAM2Long。

2. SAM2Long旨在解决SAM 2模型中的“错误积累”问题。

3. 它通过考虑每一帧中的分割不确定性，并从多个分割路径中选择最优结果。

4. SAM2Long在处理复杂长时视频时表现出色，能有效分割和跟踪对象。

5. 它在多个长时视频对象分割基准上显著提高了性能。

1. Zhè piān wénzhāng jièshào le yī zhǒng gǎijìn de shìpín duìxiàng fēngé cèlüè, chēngwéi SAM2Long

2. SAM2Long zhǐ yú jiějué SAM 2 móxíng zhōng de “cuòwù jīlěi” wènti

3. Tā tōngguò kǎolǜ měi yī zhēn de fēngé bù quèdìngxìng, bìng cóng duō gè fēngé lùjìng zhōng xuǎnzé zuì yōu jiéguǒ

4. SAM2Long zài chǔlǐ fùzá xiángshí shìpín shí biǎoxiǎn chūsè, néng yǒuxiào fēngé hé gēnzōng duìxiàng

5. Tā zài duō gè chángshí shìpín duìxiàng fēngé jīzhǔn shàng xiǎnzhù tígāo le xìngnéng

1. This article introduces an improved video object segmentation strategy called SAM2Long.

2. SAM2Long aims to address the "error accumulation" problem in the SAM 2 model.

3. It does so by considering the segmentation uncertainty in each frame and selecting the optimal result from multiple segmentation paths.

4. SAM2Long performs exceptionally well in handling complex long videos, effectively segmenting and tracking objects.

5. It significantly improves performance on multiple long-video object segmentation benchmarks.

CLIMB: CLustering-based Iterative Data Mixture Bootstrapping for Language Model Pre-training

1. 这篇文章讨论了预训练数据集通常来自网络内容，缺乏明确的领域划分。

2. 例如，广泛使用的数据集如Common Crawl没有明确的领域标签，而手动标注数据集如The Pile则需要大量劳动。

3. 为解决这些挑战，作者提出了CLIMB，一种自动化框架，能够发现、评估和优化预训练数据混合。

4. CLIMB通过嵌入和聚类大规模数据集，迭代搜索最佳混合。

5. 实验表明，使用这种混合方法训练的模型性能优于现有模型。

6. 作者还介绍了ClimbLab和ClimbMix两个数据集，并分析了最佳数据混合的特征。

1. This article discusses the fact that pre-training datasets typically come from web content and lack clear domain categorization.

2. For example, widely used datasets like Common Crawl do not have explicit domain labels, while manually annotated datasets like The Pile require significant labor.

3. To address these challenges, the authors propose CLIMB, an automated framework capable of discovering, evaluating, and optimizing pre-training data mixtures.

4. CLIMB embeds and clusters large-scale datasets, iteratively searching for the optimal mixture.

5. Experiments show that models trained using this mixture method perform better than existing models.

6. The authors also introduce the ClimbLab and ClimbMix datasets and analyze the characteristics of the optimal data mixtures.

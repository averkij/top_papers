1. This article introduces a method called Visual Reinforcement Fine-Tuning (Visual-RFT).

2. It is used to improve the performance of large vision-language models (LVLMs) in visual tasks.

3. This method works by generating multiple answers and updating the model using a verifiable reward function.

4. Experimental results show that Visual-RFT performs excellently in multiple visual tasks, outperforming supervised fine-tuning (SFT).

5. For example, in fine-grained image classification, Visual-RFT improved accuracy by 24.

6. 3%.

START: Self-taught Reasoner with Tools

1. 这篇文章介绍了一种新的大型语言模型，称为START。它通过使用外部工具和长链推理来提高复杂推理任务的能力。START能够执行复杂计算、自检、探索多种方法和自我调试。其核心创新是自学习框架，包括Hint-infer和Hint-RFT两种技术。通过这些方法，START在高级科学问答和竞赛级数学、编程基准测试中表现优异，超越了基础模型QwQ-32B，并与最先进的模型性能相当。 Zhè piān wénzhāng jièshào le yī zhǒng xīn de dàxíng yǔyán móxíng, chēngwéi START

2. Tā tōngguò shǐyòng wàibù gōngjù hé chánglián tuīlǐ lái tígāo fùzá pinyǐ rènwù de nénglì

3. START nénggòu zhíxíng fùzá suàn, zìjiǎn, tànsuǒ duōzhǒng fāngfǎ hé zìwǒ tiáoshì

4. Qí héxīn chuàngxīn shì zìxuéxí kuàngjià, bāokuò Hint-infer hé Hint-RFT liǎng zhǒng jìshù

5. Tōngguò zhèxiē fāngfǎ, START zài gāojí kēxué wèndá hé jìngsài jí shùxué, biānchéng yǐnqíng, chāoyuè le jīchǔ móxíng QwQ-32B, bìng yǔ zuì xiānjìn de móxíng xíngnéng xiāngdāng

1. This article introduces a new large language model called START.

2. It enhances the capability for complex reasoning tasks by utilizing external tools and long-chain reasoning.

3. START can perform complex computations, self-check, explore multiple methods, and self-debug.

4. Its core innovation is a self-learning framework that includes two techniques: Hint-infer and Hint-RFT.

5. Through these methods, START excels in advanced scientific question-answering and competitive-level mathematics and programming benchmarks, outperforming the baseline model QwQ-32B and matching the performance of the most advanced models.

START: Self-taught Reasoner with Tools

1. 这篇文章介绍了一种新的大型语言模型，叫做 START。

2. 它结合了外部工具来增强复杂推理任务的能力。

3. START 通过执行代码，进行复杂计算、自我检查、探索多种方法和自我调试，解决了传统大型推理模型的局限性。

4. 其创新点在于自学习框架，包括两种技术：Hint-infer 和 Hint-RFT。

5. 这些技术使模型能够高效利用外部工具，并在多个高难度基准测试中表现出色。

2. Tā jiéhé le wàibù gōngjù lái zēngqiáng fùzá pīnglǐ rènwù de nénglì

3. START tōngguò zhíxíng dàimǎ, jìnxíng fùzá jìsuàn, zìwǒ jiǎnchá, tànsuǒ duōzhǒng fāngfǎ hé zìwǒ tiáoshì, jiějué le chuántǒng dàxíng tuīlǐ móxíng de júxìanxìng

5. Zhèxiē jìshù shǐ móxíng nénggòu gāoxiào lìyòng wàibù gōngjù, bìng zài duōgè gāo nándù jīzhǔn cèshì zhōng biǎoxiàn chūsè

1. This article introduces a new large language model called START, which combines external tools to enhance its capabilities in complex reasoning tasks.

2. START addresses the limitations of traditional large reasoning models by executing code, performing complex calculations, self-checking, exploring multiple methods, and self-debugging.

3. Its innovative aspect lies in the self-learning framework, which includes two techniques: Hint-infer and Hint-RFT.

4. These techniques enable the model to efficiently utilize external tools and perform exceptionally well on multiple high-difficulty benchmark tests.

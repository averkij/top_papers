OpenCoder: The Open Cookbook for Top-Tier Code Large Language Models

1. 这篇文章介绍了大型语言模型（LLMs）在代码生成、推理任务和代理系统中的重要性。

2. 虽然开放访问的代码LLMs性能越来越接近专有模型，但适合严格科学研究的高质量代码LLMs仍然有限。

3. 为了填补这一空白，作者引入了OpenCoder，一个顶尖的代码LLMs，不仅性能媲美领先模型，还提供了完整的数据处理流水线和透明的训练协议。

4. 通过发布模型权重、推理代码、可复现的训练数据和详细的训练协议，OpenCoder旨在加速研究并实现代码AI的可复现进展。

1. This article discusses the importance of large language models (LLMs) in code generation, reasoning tasks, and agent systems.

2. While the performance of open-access code LLMs is increasingly approaching that of proprietary models, high-quality code LLMs suitable for rigorous scientific research remain limited.

3. To address this gap, the authors introduce OpenCoder, a top-tier code LLM that not only matches the performance of leading models but also provides a complete data processing pipeline and transparent training protocols.

4. By releasing model weights, inference code, reproducible training data, and detailed training protocols, OpenCoder aims to accelerate research and achieve reproducible progress in code AI.
